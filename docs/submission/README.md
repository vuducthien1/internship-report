# Hồ sơ bàn giao VDCMS

Bộ hồ sơ đáp ứng bốn nhóm yêu cầu:

1. [WORKLOG.md](./WORKLOG.md) — nhật ký các ngày phát triển có bằng chứng từ Git.
2. [PROPOSAL.md](./PROPOSAL.md) — mô tả đề tài, vấn đề, mục tiêu và lợi ích.
3. [PROJECT.md](./PROJECT.md) — hồ sơ website sau triển khai AWS và tiêu chí nghiệm thu.
4. [WORKSHOP-AWS.md](./WORKSHOP-AWS.md) — quy trình triển khai AWS từng bước và danh sách ảnh cần chụp.

## Trạng thái hiện tại

| Hạng mục | Trạng thái |
|---|---|
| Worklog | Đã dựng từ lịch sử Git; cần bổ sung thành viên và số giờ thực tế |
| Proposal | Đã soạn |
| Project source code | Hoàn thành và kiểm thử local |
| Infrastructure as Code | Đã có CloudFormation và script deploy |
| Website chạy thật trên AWS | **Chưa provision**; cần AWS account và chấp nhận chi phí |
| Workshop | Đã có quy trình; ảnh chụp thật cần bổ sung trong lúc deploy |

Không điền URL, ảnh hoặc kết quả AWS giả. Chỉ chuyển trạng thái sang hoàn tất sau khi stack thật hoạt động và đã chạy checklist nghiệm thu.
