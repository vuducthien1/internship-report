- **Quyền hạn tài khoản:** Sử dụng tài khoản AWS có phân quyền gốc (`AdministratorAccess`) hoặc quyền quản trị được ủy quyền để đảm bảo khả năng tạo, chỉnh sửa và gỡ bỏ Stack mà không bị chặn bởi các chính sách IAM.
- **Môi trường CloudShell:** Công cụ dòng lệnh chạy trên nền tảng trình duyệt Web có cài đặt sẵn AWS CLI phiên bản mới nhất, Git, Node.js và NPM để thực hiện cài đặt các gói phụ thuộc (dependencies).
